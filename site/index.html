View on GitHub

The problem, if you can call it that

Claude Code agents read files, edit code, run tests. Important stuff. And the way you experience all of this is: text. Scrolling. Forever.

You could read it. Or you could watch pixel people walk to their desks instead. We built the second thing.

What you get

👥

6 permanent employees

They idle in the lounge until you give them work. They wander near the arcade machines. They do not play ping pong.

🚪

Temp contractors

Show up when your staff can't keep up. Do their job. Leave through the top door without saying goodbye.

👔

A CEO

Sits alone in a blue office. When a new task arrives, sprints to the whiteboard and yells. The most realistic part.

🖥️

Animated monitors

Rainbow pixels that shift every few ticks. Your agents are definitely working and not browsing Reddit.

🕹️

Arcade machines

In the lounge. They only turn on when someone walks up. Neon colors. Very retro. Very distracting.

📊

Performance stats

FPS and RAM in the status bar. Because you should monitor the performance of your monitoring tool.

Setup

You need Rust and a tolerance for whimsy.

git clone https://github.com/anduong96/agents-story.git
cd agents-story
cargo build --release
cargo run -- --demo

That's it. We didn't say it was hard. We said it was pointless.

MIT — because restricting access to something this unnecessary felt wrong.
